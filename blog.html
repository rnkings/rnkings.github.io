HTML, CSS & JavaScript explained.


HTML

HTML or HyperText Markup Language and uses tags to identify content and delivers basic structure to websites.

CSS

CSS or Cascading Style Sheets are what gives a website its style.

JavaScript

JavaScript controls the behavior of a website. It's a programming language used by web developers to design interactive websites. One example would be pop up windows.

Hmmm what does all of this mean?

Think of building a website as you would build a house. HTML is the basic structure or frame of the house 2x4s, drywall etc. CSS is the house with added siding, interior paint colors, decor etc. You would be able to change any of those things without changing the structure. Adding JavaScript adds function to the house like gas, electric and plumbing.

Best Websites to Learn Code

In this post I will have two categories of sites: free and not free. From there I will break down the pros and cons of each site.

Free Options

Code Academy:

Pro: Code Academy is a free site that walks you through small exercises that you type out while you learn. This site is good for learning the basics.
Con: If you learn best by watching videos Code Academy might not be the best way for you to learn code. While it is interactive and has exercises you can type as you go along, it is mostly a read and do site. In my opinion Code Academy is a good place to start but it can only get you so far and doesn’t have as much information to offer as other sites.

Youtube:

Pro: Youtube is free and has a wealth of knowledge to sift through. If you don’t like how one person teaches, you can find someone else that will teach you the same thing. Youtube is also a great source if you are stumped and need to look up how to do a specific thing instead of a general learning video. Quentin Watt has an amazing Youtube channel that has courses in HTML, CSS, JavaScript, Bootstrap and more. He has updated his HTML and CSS to the new standard and tries to keep his videos up to date.
Con: Old content. People who upload their videos to youtube don’t always take down out of date information. You have to be careful in selecting the most current videos that practice current code practices. The developer world is fast paced and constantly changing. So make sure you can keep up or your skills won’t be as valuable.

Not Free Options

Treehouse:

Pro: Treehouse is my personal favorite site to learn code. It costs 25 dollars a month and has a wide variety of courses that go pretty well in depth. Treehouse does a good job of explaining things step by step with an instructor in videos that have code alongs and quizzes.
Con: Treehouse is done well but since a lot of work goes into making a new course they have not yet added newer languages such as React even though it’s been requested by popular demand.

Lynda.com:

Pro: At Lynda.com you can learn more than just coding. It’s a wealth of knowledge with videos on Adobe Illustrator, Photoshop and video editing. They also have small projects they walk you through that you might want to use on your current website, like a rotating carousel with a jQuery library they provide you.
Con: There are so many videos on Lynda.com some are bound to be old. One video I found was using a four year old jQuery library that was not being maintained anymore and was prone to breaking.
